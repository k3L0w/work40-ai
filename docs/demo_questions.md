# Demo Questions — Assistente RAG

Use estas perguntas para demonstrar o chatbot com respostas em portugues e fontes
internas.

## Perguntas principais

- Como devo me preparar para a Industria 4.0?
- Quais competencias devo priorizar para trabalhar com dados industriais?
- Como a automacao pode impactar tarefas repetitivas no meu cargo?
- Que habilidades humanas continuam importantes com IA e automacao?
- Como montar um portfolio para uma carreira em automacao e dados?

## Perguntas para estudantes

- Sou estudante e quero entrar na area de dados industriais. Por onde comeco?
- Quais projetos pequenos posso fazer para demonstrar competencias Work4.0?
- Como equilibrar Python, dados e comunicacao no meu plano de estudos?

## Perguntas para trabalhadores

- Trabalho em operacoes. Como posso me preparar para automacao sem discurso alarmista?
- Quais tarefas devo mapear antes de aprender ferramentas de IA?
- Como transformar experiencia operacional em portfolio digital?

## Perguntas para professores

- Como explicar Industria 4.0 para uma turma tecnica?
- Que atividade pratica posso propor sobre automacao e competencias humanas?
- Como avaliar um projeto de aprendizagem sobre IA e trabalho?

## Perguntas para empresas/RH

- Como identificar oportunidades de reskilling em uma equipe operacional?
- Quais cuidados devo ter antes de automatizar processos?
- Quais indicadores simples posso acompanhar em um piloto de automacao?

## Pergunta de baixa confianca

Use uma pergunta fora do escopo para demonstrar o guardrail:

- Qual e a melhor receita medieval para pao com especiarias?

Resultado esperado: o app deve informar que nao encontrou informacao suficiente
na base de conhecimento para responder com precisao.
