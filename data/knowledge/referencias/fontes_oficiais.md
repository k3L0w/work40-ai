# Fontes oficiais e referências recomendadas

## Resumo executivo

Esta nota lista fontes confiáveis para atualização da base Work4.0 AI. As notas devem ser originais, curtas, parafraseadas e sempre acompanhadas de fontes consultadas.

## Conceitos principais

- Fontes oficiais reduzem risco de informação frágil.
- Relatórios devem ser transformados em notas aplicáveis.
- Citações devem registrar organização e URL.
- A base precisa ser revisada periodicamente.

## Aplicações práticas

- Atualizar notas com novas edições de relatórios.
- Adicionar fontes nacionais para contexto brasileiro.
- Separar conceito, aplicação e risco.
- Criar notas por tema para melhorar recuperação RAG.

## Impactos no trabalho

- Aumenta credibilidade do assistente.
- Ajuda usuários a entender limites das respostas.
- Facilita apresentação para SENAI, GitHub e recrutadores.

## Competências relacionadas

- curadoria
- pesquisa
- síntese
- ética
- documentação

## Riscos e cuidados

- Copiar longos trechos de fontes externas.
- Usar blogs sem credibilidade como base principal.
- Manter URLs sem revisão.

## Como se preparar

- Revisar fontes a cada fase do projeto.
- Preferir fontes institucionais.
- Registrar data de revisão no frontmatter.
